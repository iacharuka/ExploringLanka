New Astro starter for ExploringLanka

Quick start:

1. Copy images from the original project's `assets/images/` into `new-site/public/assets/images/`.
2. Install dependencies:

```bash
cd new-site
npm install
npm run dev
```

This starter contains a simple `Header.astro`, `Footer.astro`, and `src/pages/index.astro` with the homepage content ported.
